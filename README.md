# smart-expense-tracker
A web-based expense tracking application built using Vanilla JavaScript.
<br>
<br>
---

<h2>Features</h2>
<pre>-Add income and expenses
-Category-based tracking
-Persistent storage using LocalStorage
-Analytics dashboard with charts
</pre>

---

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
- Chart.js

---

## Learning Outcomes
- DOM manipulation using JavaScript
- State management with arrays and objects
- Client-side data persistence using LocalStorage
- Data analytics and visualization
- Writing clean and modular JavaScript code

---

## How to Run the Project
1. Clone or download the repository  
2. Open `index.html` in any modern web browser  
3. Start adding income and expense transactions  

---

## Purpose
This project was built to strengthen core JavaScript fundamentals by developing a real-world application without using frameworks.
